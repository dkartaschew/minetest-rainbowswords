Rainbow Swords
==========

D.Kartaschew
For Minetest 0.4.14 and later

### Depends: 
* default
* dye.

## Description

Adds coloured swords to the game.

The resulting swords have the same properties as the underlying steel sword.

Crafting
--------

Coloured Sword x1
```
SS dye
```
SS = Steel Sword

Ghost Sword
```
DS blue dye
```
DS = Diamond Sword
